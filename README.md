# room-booking
Сервис бронирования комнат
### Сборка и запуск через Docker Compose
1. Клонируйте репозиторий и перейдите в директорию проекта:
```sh
git clone https://github.com/internships-backend/test-backend-GeorgiiPoliakov1.git
cd test-backend-GeorgiiPoliakov1
```
2. Соберите и запустите сервис:
```sh
docker compose up --build
```
